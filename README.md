# To Do List App

A simple and responsive To Do List application built with React.

## Live Demo

🔗 https://to-do-list-gamma-beige.vercel.app/

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Clean and simple user interface
* Responsive design

## Built With

* React
* JavaScript (ES6+)
* CSS

## Getting Started

Clone the repository:

```bash
git clone https://github.com/marwanibrahimomran/To-do-list.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
